# แบบฝึกหัด Git

ชื่อ-นามสกุล : อัมพิรา นันตา

ห้อง : ปวช 2/2

เลขที่ : 14

----------------------------------

ข้อที่ 1
Git คืออะไร

ตอบ : git คือ Version Control System (VCS)   ที่ใช้สำหรับการติดตามของไฟล์ โดยเฉพาะไฟล์โค้ด โปรแกรมช่วยให้นักพัฒนาสามาถรบันทึกประวัติการแก้ไข ทำงานร่วมกันหลายคนและย้อนกลับไปยังเวอร์ชั่นก่อนหน้าได้

----------------------------------

ข้อที่ 2
Repository คืออะไร

ตอบ : Repository (Repo) คือพื้นที่สำหรับเก็บโปรเจกและไฟล์ทั้งหมด รวมถึงประวัติการแก้ไขที่ Git บันทึกไว้ เปรียบเสมือน "โฟลเดอร์อัจฉริยะ" ที่สามารถติดตามการเปลี่ยนแปลงของไฟล์ได้

----------------------------------

ข้อที่ 3
คำสั่งใดใช้ Clone Repository

ตอบ : คำสั่งที่ใช้ Clone Repository คือ  git clone https://github.com/username/myproject.git

----------------------------------

ข้อที่ 4
คำสั่งใดใช้เพิ่มไฟล์เข้าสู่ Staging Area

ตอบ : คำสั่งที่ใช้เพิ่มไฟล์เข้าสู่ Staging Area คือ git add , เพิ่มไฟล์เดียว git add index.html , เพิ่มทุกไฟล์ที่มีการเปลี่ยนแปลง git add .

----------------------------------

ข้อที่ 5
คำสั่งใดใช้บันทึกการเปลี่ยนแปลง

ตอบ : คำสั่งที่ใช้ บันทึกการเปลี่ยนแปลง (Commit) คือ git commit -m "ข้อความอธิบายการเปลี่ยนแปลง"

----------------------------------

ข้อที่ 6
คำสั่งใดใช้ส่งข้อมูลขึ้น GitHub

ตอบ : คำสั่งคือ git push (หรือ git push origin main)

----------------------------------

ข้อที่ 7
วันนี้ได้เรียนรู้อะไรเกี่ยวกับ Git
ตอบ : วันนี้ได้เรียนรู้เกี่ยวกับ Git ซึ่งเป็นระบบควบคุมเวอร์ชัน (Version Control System) ที่ใช้สำหรับติดตามและบันทึกการเปลี่ยนแปลงของไฟล์ ช่วยให้สามารถย้อนกลับไปยังเวอร์ชันก่อนหน้าและทำงานร่วมกับผู้อื่นได้อย่างมีประสิทธิภาพ

----------------------------------
